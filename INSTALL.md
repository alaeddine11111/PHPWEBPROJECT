# Installation Guide

## Quick Setup with WAMP

### 1. Download and Install
1. Install [WAMP Server](https://www.wampserver.com)
2. Clone this repository to: `C:\wamp64\www\PHPWEBPROJECT\`

### 2. Database Setup
1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Create database: `gestion_stocke`
3. Import `gestion_stocke(4).sql` 

### 3. Run the Application
1. Start WAMP services
2. Open: http://localhost/PHPWEBPROJECT
3. Login with default credentials
